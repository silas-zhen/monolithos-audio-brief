## What It Does

Select any note in your vault. Choose a mode. Press TRANSMIT.
Audio Brief generates a full podcast-style conversation — two AI hosts
analyzing, debating, and riffing on your content.

The audio plays right inside Obsidian. Download it. Listen on your commute.
Share it with your team. A full transcript auto-saves as `.md` in your vault.

## Install

### From Obsidian Community Plugins
1. Open Settings → Community Plugins → Browse
2. Search "Audio Brief"
3. Install and enable
4. Enter your API key in plugin settings

### Manual Install
1. Download the latest release from [Releases](../../releases)
2. Extract to `your-vault/.obsidian/plugins/audio-brief/`
3. Enable in Settings → Community Plugins

---

## How It Works

```
Your Note → AI Script Generation → Professional Voice Synthesis → Audio in Vault
```

1. Select one or more notes as source material
2. Choose your podcast mode
3. Choose output language
4. Press TRANSMIT
5. Listen. Download. Share.

Processing takes 2-8 minutes depending on content length and mode.
A transcript is automatically saved as a `.md` file in your vault.
